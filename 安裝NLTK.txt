
在Windows 11上安裝NLTK（Natural Language Toolkit）很簡單。以下是安裝步驟：

安裝Python：
如果您尚未安裝Python，請先從Python官方網站下載並安裝最新版本的Python。安裝時請確保勾選"Add Python x.x to PATH"（將Python添加到系統路徑中）的選項。
打開命令提示符：
在Windows 11中，按下Win + R組合鍵打開運行對話框，然後輸入"cmd"並按Enter，即可打開命令提示符。
使用pip安裝NLTK：
在命令提示符中，輸入以下命令來使用pip安裝NLTK：
pip install nltk

下載NLTK資源：
安裝完成後，您需要下載NLTK所需的資源。您可以選擇下載全部資源或僅下載特定的資源。執行以下Python命令來下載全部資源：
import nltk
nltk.download('all')
